# livestreamlab.live domain mapping

This deployment is independent and does not rely on Base44.

## Primary platform

- App: https://livestreamlab.live

## Service domains (port 8080)

- API: https://api.livestreamlab.live:8080
- Auth: https://auth.livestreamlab.live:8080
- Uploads: https://uploads.livestreamlab.live:8080
- CDN: https://cdn.livestreamlab.live:8080
- WebSocket: wss://ws.livestreamlab.live:8080

## DNS records to configure

- api.livestreamlab.live
- auth.livestreamlab.live
- uploads.livestreamlab.live
- cdn.livestreamlab.live
- ws.livestreamlab.live
